业余做的一些东西

  • 音乐笔记+：记录灵感乐句；一个人也可以像乐队般练习；精准的节拍器，练琴必备。
  • 为你搜谱：一个快速简洁的吉他谱检索网站。
  • JApiDocs: 给 Java 程序员用的 Api 文档生成工具。
  • SqliteLookup：可以在 Android 手机上查看和编辑 sqlite.db 的数据库查看工具。
  • Android-YRichEditor：Android 原生富文本编辑器。
  • SQLiteUtils：sqlite 简单的 orm 封装，简化 sqlite 的使用，有 GreenDao 等更好的库了，不推荐使用。
  • Java For Kids：翻译的一本给小孩的 Java 编程书。
