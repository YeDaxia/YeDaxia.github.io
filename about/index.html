一些瞎折腾

  1. 为你搜谱：一个快速简洁的吉他谱检索网站。
  2. JApiDocs: 给 Java 程序员用的 Api 文档生成工具。
  3. SqliteLookup：可以在 Android 手机上查看和编辑 sqlite.db 的数据库查看工具。
  4. Android-YRichEditor：Android 原生富文本编辑器。
  5. SQLiteUtils：sqlite 简单的 orm 封装，简化 sqlite 的使用，有 GreenDao 等更好的库了，不推荐使用。
  6. Java For Kids：翻译的一本给小孩的 Java 编程书。
