《首页 用户接口:

  • 用户登录
  • 修改用户资料

用户登录

method：POST
url: dev.api.yiyoushuo.com/userLogin

参数列表

参数名 类型 必需 描述
nameStringtrue用户名
passwordStringtrue密码

返回结果: Android Code | IOS Code

{
  "id": "String //用户id",
  "token": "String //用户凭证",
  "name": "String //名字",
  "avatar": "String //头像"
}

修改用户资料

method：POST
url: dev.api.yiyoushuo.com/userModify

参数列表

参数名 类型 必需 描述
nameStringfalse用户名
avatarStringfalse头像

返回结果: Android Code | IOS Code

{
  "id": "String //用户id",
  "token": "String //用户凭证",
  "name": "String //名字",
  "avatar": "String //头像"
}
